# dinnerpickr

🍝 Meal Picker App

A simple web app that helps you decide what to eat for dinner in seconds.

Instead of spending time thinking “what should we eat?”, you can add your own meals, organize them into categories, and let the app randomly pick one for you.

✨ Features
🗂️ Create custom meal categories
🍲 Add your own meals and tags
🎲 Smart random meal selection
📜 History of previous picks
🔐 User accounts with Supabase login
☁️ Cloud saving (no local storage needed)
🌍 Available in English & Danish
💡 Why I built it

I always ran into the same problem: deciding what to eat for dinner took longer than actually cooking.

I wanted something simple that could:

remove decision fatigue
use my own meals
make the choice instantly

So I built this tool to solve that problem for myself — and decided to share it in case others feel the same way.

🚀 How it works
Create an account / log in
Add your meals and categories
Click “Pick meal”
Get a random suggestion instantly
